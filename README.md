# Borion-Hardware-Store
Elektronik donanım ürünleri için geliştirdiğim, sepet ve kullanıcı giriş mantığı içeren e-ticaret arayüzü.

# 🛒 Borion E-Ticaret Platformu

Bu proje, bir donanım mağazası için sıfırdan tasarlayıp kodladığım bir e-ticaret web arayüzüdür. Amacım, gerçek bir e-ticaret sitesindeki kullanıcı deneyimini (UI/UX) ve sayfa yönlendirmelerini başarılı bir şekilde koda dökmektir.

## 🚀 Özellikler
* Ürün kategori filtreleme (Ekran Kartı, RAM, İşlemci vb.)
* Ürünleri sepete ekleme ve sepet tutarı hesaplama mantığı
* Kullanıcı giriş (Login) ekranı simülasyonu
* İletişim formu ve modern sayfa tasarımı

## 🛠️ Kullanılan Teknolojiler
* HTML5
* CSS
* JavaScript
